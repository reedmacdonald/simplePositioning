Positioning Divs from Simple to Complex

In the following, we will look at the properties of "display": block, inline, and inline-block. Block and inline block with have adjustable heights. Inline and inline blocks will have a default width of the inner text, while block will have a default width of the parent element. Both width's will be a percentage of the parent's width if set as a percentage. Also block adds a natural line-break (interesting!)

Now on to display:flex. This is it's default layout, with the main axis being horizontal, with it's flex direction being row. The most important property is justify-content. This organizes the content along the main axis (in this case, horizontal). Align-items is the opposite of justify content, and works on the other axis. This is annoying because you'd intuitively think that align-content would be the opposite of justify-content, but, unfortunately, it is not. Also, the flexbox guide on csstricks only goes through justify-content, align-items, align-content, and justify-self, so I'm just going to pretend those other properties like justify-self don't exist. Also, last thing, it seems like this makes all of the items inline-bloc, but if you don't do an align-items they stretch to fill their cross-axis so they look bigger.
